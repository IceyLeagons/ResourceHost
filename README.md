# ResourceHost
A stupidly simple free open-source temporary resource pack hosting based on Spring.

WIP (in Beta phase)!
Please do not use this software in production just yet!
There may be some security/DoS issues, but we've started testing it ourselves. Some other features are still in the making as well, and they may not work as expected!
SSL support is planned, but not yet implemented!

The public instance will only be revealed after we've met our goals.
